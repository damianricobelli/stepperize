---
title: Open in v0
description: Registry examples with Radix UI or Base UI. Switch tabs to preview and copy the code for each.
---

Each example is available in two variants. Use the tabs to choose **Radix UI** or **Base UI**, then open in v0 or run the `shadcn add` command to copy.

## Stepper demo

<Tabs groupId="stepper-demo" items={["Radix UI", "Base UI"]} defaultIndex={0}>
  <Tab value="Radix UI">
    <OpenInV0 name="stepper-demo" registry="radix-ui">
      <StepperDemo />
    </OpenInV0>
    ```bash

    ```
  </Tab>

</Tabs>
