---
title: Utils
description: Pure utility functions for working with steps
---

The `utils` object returned by `defineStepper` provides pure functions over the step array. They do not depend on React or current state.

## Functions

| Function | Description | Returns |
| -------- | ----------- | ------- |
| `getAll` | All steps | `Steps` |
| `get(id)` | Step by ID | `Get.StepById<Steps, Id>` or `undefined` |
| `getIndex(id)` | Index of step by ID | `number` |
| `getByIndex(index)` | Step at index | `Steps[Index]` |
| `getFirst` | First step | `Steps[number]` |
| `getLast` | Last step | `Steps[number]` |
| `getNext(id)` | Next step after `id` | `Steps[number]` |
| `getPrev(id)` | Previous step before `id` | `Steps[number]` |
| `getNeighbors(id)` | `{ prev, next }` for step `id` | `{ prev: Steps[number] \| null; next: Steps[number] \| null }` |

There is no `has` or `count`; use `utils.get(id) != null` and `utils.getAll().length` if needed.

## Usage

```tsx twoslash
import * as React from "react";
import { defineStepper } from "@stepperize/react";

const { utils, useStepper } = defineStepper(
  { id: "step1", title: "Step 1" },
  { id: "step2", title: "Step 2" },
  { id: "step3", title: "Step 3" }
);

function StepNavigator() {
  const stepper = useStepper();

  const allSteps = utils.getAll();
  const currentIndex = utils.getIndex(stepper.current.id);
  const firstStep = utils.getFirst();
  const lastStep = utils.getLast();
  const stepById = utils.get("step2");
  const nextStep = utils.getNext(stepper.current.id);
  const { prev, next } = utils.getNeighbors(stepper.current.id);

  return (
    <div>
      <p>Step {currentIndex + 1} of {allSteps.length}</p>
      <p>Current: {stepper.current.title}</p>
      {next && <button onClick={() => stepper.goTo(next.id)}>Go to {next.title}</button>}
    </div>
  );
}
```
